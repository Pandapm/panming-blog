    浏览器一般有三类 web Worker：

    • Worker：专用的 worker，只能被创建它的 JS 访问，生命周期到创建它的页面关闭时结束。
    • SharedWorker：共享的 worker，可以被好几个 JS 访问，生命周期到关联的页面都关闭时结束。
      ServiceWorker：一个特殊的 worker，生命周期与页面无关，关联页面未关闭时，它也可以退出，没有关联页面时，它也可以启动
